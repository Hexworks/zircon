class CP437TileMetadataLoader(width: Int, height: Int) : TileMetadataLoader<CharacterTile>
Content copied to clipboard

This class can be used to look up the metadata for the Chars supported by Code Page 437. CP437 has a fixed grid (16x16) of characters so its metadata can be pre-calculated. You can access it through the cp437Metadata field.
