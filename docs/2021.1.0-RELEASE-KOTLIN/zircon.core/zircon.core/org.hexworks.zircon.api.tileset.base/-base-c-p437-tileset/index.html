BaseCP437Tileset

abstract class BaseCP437Tileset<S : Any, T : Any>(resource: TilesetResource, targetType: KClass<S>) : Tileset<S>
Content copied to clipboard

This base class can be used to implement CP437 tilesets. You are not required to use this, it was implemented for convenience only. This implementation contains all the checks and lookups that are necessary for CP437. What you need to implement:

  • drawTile: this is the actual draw operation

  • loadTileTexture: this is the loading of a texture for a given tile. What you don't need to implement

  • Checking for CharacterTile

  • Caching

  • Creating the necessary metadata for the texture

Parameters

S

the class of the surface that we'll draw upon (Graphics2D for Swing for example)

T

the class of the individual texture tile that can be cached (BufferedImage for Swing for example)
